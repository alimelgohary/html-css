            /* General styling for all examples */
            .parent-one, .parent-two, .parent-three {
                margin-bottom: 25px;
                padding: 10px;
                background-color: #f7f7f7;
            }

            /* 1. Default Inheritance */
            .parent-one {
                font-size: 30px;
                color: green;
            }
            .parent-one p, .parent-one span {
                margin: 5px 0;
            }

            /* 2. Non-Inherited Properties */
            .parent-two {
                border: 5px solid red;
                padding: 10px;
            }
            .child-two {
                margin-top: 10px;
                padding: 5px;
                background-color: #ddd;
                margin-left: 20px;
            }

            /* 3. Forcing Inheritance with 'inherit' */
            .parent-three {
                border: 3px dashed blue;
                padding: 10px;
            }
            .child-three {
                border: inherit; /* Explicitly forces the child to take the parent's border */
                margin-top: 10px;
                padding: 5px;
                background-color: #ddd;
            }

1. Demonstrating Default Inheritance (e.g., color, font-size)

This parent has a large font-size and green color (defined in .parent-one).

The paragraph inherits the font-size and color.

The span also inherits the font-size and color.

2. Demonstrating Non-Inherited Properties (e.g., border, padding)

This parent has a red border and padding.
This child does **not** inherit the border by default, showing that properties affecting the box model must be set explicitly.

3. Forcing Inheritance using the inherit Keyword

This parent has a dashed blue border.
This child explicitly uses border: inherit; to get its parent's border style.
