1. Setting Text Slant with font-style

The font-style property is used to create slanted text, typically using the value italic.

This text is set to italic using font-style: italic;.

It can also be used to remove an existing slant, such as from an <i> tag.

This italic text is forced back to normal with font-style: normal;.

2. Transforming Text to small-caps

The font-variant: small-caps; property renders lowercase letters as small capital letters.

This text uses font-variant: small-caps.

3. Controlling Boldness with font-weight Keywords

The font-weight property controls the weight or thickness of the text. Keywords like bold and normal are commonly used.

This text uses font-weight: bold;.

Like font-style, it can reset the default weight of elements like <b> or <h1>.

This bold text is forced back to normal with font-weight: normal;.

4. Controlling Boldness with font-weight Numeric Values (100-900)

Numeric values (100, 200, ..., 900) offer fine-grained control over boldness, provided the font supports the specific weight.

This text uses font-weight: 900; (often Black or Heavy).

This text uses font-weight: 200; (often Extra-Light or Thin).
