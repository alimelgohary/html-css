1. Understanding em Units (Relative to Parent)

The em unit is relative to the font size of its **parent element**. If the parent is 20px, 1.5em will be 30px.

Parent is 20px

This text is 1.5em (30px) because the parent is 20px.

2. Understanding rem Units (Relative to Root)

The rem unit is relative to the font size of the **root element** (html). By default, 1rem is 16px, regardless of the immediate parent's size.

Parent is 20px

This text is 2rem (32px) regardless of the parent, based on the default root size (16px).

3. Using Percentage Units (Relative to Parent)

Percentage units behave similarly to em, setting the size relative to the immediate **parent element**'s font size.

Parent is 20px

This text is 150% (30px) of the parent's font size.

4. Using Viewport Width Unit (vw) for Scalable Text

The vw unit is relative to the **viewport width** (1vw = 1% of the viewport width). This allows text size to scale dynamically as the user resizes the browser window.

This Heading Scales with the Browser Width (5vw).
