1. Using Serif and Sans-Serif Generic Families

Using the final generic family fallback (serif or sans-serif) ensures a basic font style is always loaded, even if the specified named fonts fail.

This text uses a Serif font family (Times, "Times New Roman", serif).

This text uses a Sans-Serif font family (Arial, Verdana, sans-serif).

2. Demonstrating the Font Fallback System

The browser reads the comma-separated list and uses the first font it finds. Here, it will skip "NonExistentFont" and use "Courier" or the generic "monospace".

This text demonstrates the font fallback list: "NonExistentFont", Courier, monospace.

3. Using a Custom Font via Google Fonts (Roboto)

For custom web fonts like those from Google Fonts, you must first link the font file in the HTML head (using the "Roboto" example here) before applying it in CSS.

This text is styled with the Roboto font family, loaded from Google Fonts.
