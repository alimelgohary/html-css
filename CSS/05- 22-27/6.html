2. Demonstrating Parent Collapse (The Float Problem)

When children are floated, they are removed from the normal flow. The parent element (red border, yellow background) acts as if they don't exist, and its height collapses. The paragraph below flows around the floated box.

Floated Box

This paragraph flows right under the box because the parent's height collapsed to zero.

3. Fixing Layout with a Sibling Clear Element (clear: both;)

Inserting an empty element with clear: both; after the last floated element forces the document flow to restart, pushing the content below the floated boxes.
