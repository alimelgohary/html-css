1. pointer and help (Indicating Action)

Use pointer (the hand icon) for clickable elements, and help (the question mark) when hovering suggests assistance is available.

Hover for Pointer (Clickable)
Hover for Help (?)

2. not-allowed and wait (Indicating State)

The not-allowed cursor indicates a forbidden action, while wait indicates that the system is busy processing a task.

Hover for Not-Allowed
Hover for Wait (Processing)

3. move and Dragging Cursors

These cursors are used to indicate that an element can be freely moved (move) or is actively being dragged (grab and grabbing).

Hover for Move
Hover for Grab (Ready to Drag)
Hover for Grabbing (Actively Dragging)

4. Resizing Cursors

Used to indicate that an element can be resized along a specific axis or direction.

  • ew-resize: East-West (Horizontal) resizing.
  • nesw-resize: North-East/South-West (Diagonal) resizing.
Hover for East-West Resize
