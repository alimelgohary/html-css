02-Spacing & Borders/Outline

When adding a border, border&element is considered the new element

In Outline, element stays in place while outline goes outside of it

Outline can't change each direction color|style|weight (it can be only one value)

Outline is not widely used

            
            .div-1 {
                background-color: yellow;
                padding: 20px;
                width: 500px;
                outline: 10px red solid;
            }
            .div-2 {
                background-color: yellow;
                padding: 20px;
                width: 500px;
                border: 10px red solid;
            }
