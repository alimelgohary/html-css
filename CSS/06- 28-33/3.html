Adding margin to an element

Adding margin to an element changes its position and also other elements under it change

One
Two

Position Relative

Element has a space in the layout

When you specify top:0px or bottom: or left: or right: the element moves relative to its intial positon regardless of other elements

One
Two

Position absolute

Element one has NO space in the layout

When you specify top:0px or bottom: or left: or right: the element moves relative to its root parent element

One
Two

Position absolute with parent

Element one has NO space in the layout

Parent element must have position relative

When you specify top: or bottom: or left: or right:0px the element moves relative to its parent element

One
Two

Position fixed

Element one has NO space in the layout

Element position is relative to top root element

Element also doesn't move when you scroll, can be used to implement scroll to top feature

When you specify top: or bottom: or left: or right:0px the element moves relative to its parent element

Two

Position sticky

Element has Regular space in the layout

When you specify top:0 or bottom:0 or left: or right: the element sticks when visible part of the screen reaches that point
