1. Pure Percentage Distribution (Without calc())

This is the initial approach. For 4 items and 5 margins (before, between, after) each at 1%, the total margin is 5%. Remaining width is 95%. Each item gets 95% / 4 = 23.75%.

2. Calculating 4-Item Width with Fixed Pixel Margins using calc()

This is the key application: mixing units. With 4 items, we need 5 margins. If each margin is a fixed 10px, the total margin is 50px. The width is calculated as calc((100% - 50px) / 4), guaranteeing perfect alignment regardless of container width.
