    /* Essential Properties for the Flip Effect */

    .card {
        transform-style: preserve-3d; /* Keep children in 3D space */
        transition: transform 0.8s;
    }

    .container:hover .card {
        transform: rotateY(180deg); /* Flip the card */
    }

    .face {
        backface-visibility: hidden; /* Hide face when turned away */
    }

    .back {
        transform: rotateY(180deg); /* Back face starts reversed */
    }

Practical Example: Flip Product Card

Hover over the card below to see the 3D flip effect in action. This uses preserve-3d to ensure the front and back faces move together as a solid object [00:05:04].

Key Concepts from the Video:

  • backface-visibility: hidden; Prevents the "mirror image" of the front face from being visible when the element is rotated 180 degrees, allowing the back face to show through instead [00:04:51].
  • transform-style: preserve-3d; This is applied to the parent. Without it, the children (front and back faces) would be flattened into the plane of the parent and wouldn't maintain their 3D positions during the rotation [00:05:04].
  • The Setup: The "Back" face must be pre-rotated by 180deg [00:03:55]. When the main container is rotated on hover [00:04:16], the Front face moves to 180deg (becoming hidden) and the Back face moves to 360deg (becoming visible).
