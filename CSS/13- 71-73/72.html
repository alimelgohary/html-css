    /* 1. Perspective on Parent (Essential for 3D) */
    .container {
        perspective: 400px;
    }

    /* 2. TranslateZ (Closer to viewer) */
    .box-z:hover {
        transform: translateZ(100px);
    }

    /* 3. Translate3d(x, y, z) */
    .box-3d:hover {
        transform: translate3d(50px, 20px, 100px);
    }

    /* 4. Perspective Origin (Vanishing Point) */
    .container-origin {
        perspective-origin: top left;
    }

1. Positive `translateZ` (Zoom In effect)

Moving an element on the Z-axis toward the viewer (positive value). It appears larger as it "comes out" of the screen toward you.

TranslateZ (100px)

2. Negative `translateZ` (Zoom Out effect)

Moving an element on the Z-axis away from the viewer (negative value). It appears smaller as it recedes into the screen.

TranslateZ (-100px)

3. Using `translate3d(x, y, z)`

A shorthand function that allows moving the element along all three axes simultaneously. This example moves it right, down, and toward the viewer.

Translate3d

4. Understanding `perspective-origin`

The perspective-origin property sets the position of the viewer's eye. By setting it to top left, the element appears to move toward that corner as it scales up, changing the vanishing point of the 3D space.

Perspective Origin
