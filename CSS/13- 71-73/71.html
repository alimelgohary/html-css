1. Rotation Around the X-Axis (`rotateX`)

The rotateX() function rotates an element around the horizontal X-axis. This creates a tilting effect where the element flips forward or backward in 3D space [00:03:06].

Rotate X (180deg)

2. Rotation Around the Y-Axis (`rotateY`)

The rotateY() function rotates an element around the vertical Y-axis. This is commonly used for "card flip" effects, where the element spins like a revolving door [00:03:06].

Rotate Y (180deg)

3. Rotation Around the Z-Axis (`rotateZ`)

The rotateZ() function rotates the element around the Z-axis (the axis pointing directly at the viewer). In a 2D context, this is identical to the standard rotate() function [00:01:55].

Rotate Z (45deg)

4. Vector Rotation using `rotate3d()`

The rotate3d(x, y, z, angle) function allows for rotation around a custom vector in 3D space. The first three values are coordinates (0 to 1) defining the vector, and the fourth is the angle of rotation [00:03:34].
