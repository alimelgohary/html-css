    }

CSS Filters (filter Property)

The filter property applies graphical effects like blurring or color shifting to an element. It is often used on images, but can be applied to any element.

Note: For these examples to display correctly, you need to replace placeholder.jpg with a local image file.

1. grayscale() Filter

Converts the image to black and white. Try hovering over the image.

2. blur() Filter

Applies a Gaussian blur to the image. Try hovering over the image to remove the blur.

3. invert() Filter

Inverts the colors of the image. Try hovering over the image for a partial inversion effect.
