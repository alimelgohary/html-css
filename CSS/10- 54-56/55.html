    /* 1. Default Direction */
    .ex-default {
        background-image: linear-gradient(red, blue); /* 55-1: Defaults to 'to bottom' */
    }

    /* 2. Direction Keywords */
    .ex-direction {
        background-image: linear-gradient(to right, green, yellow); /* 55-2: Goes from left to right */
    }

    /* 3. Angles */
    .ex-angle {
        background-image: linear-gradient(45deg, black, purple); /* 55-3: 45 degree angle */
    }

    /* 4. Color Stops (Smooth Transition) */
    .ex-stops {
        background-image: linear-gradient(to right, 
            red 20%, /* Red stops at 20% */
            blue 80% /* Blue starts at 20% and stops at 80% */
        ); 
    }

    /* 5. Border Effect (Sharp Color Transitions) */
    .ex-border {
        background-image: linear-gradient(to right,
            #007bff 20%, /* Blue ends at 20% */
            #28a745 20%, #28a745 40%, /* Green starts at 20%, ends at 40% */
            #ffc107 40%, #ffc107 60%, /* Yellow starts at 40%, ends at 60% */
            #dc3545 60%, #dc3545 80%, /* Red starts at 60%, ends at 80% */
            purple 80% /* Purple starts at 80%, ends at 100% */
        ); /* 55-5: Creates sharp divisions */
    }

CSS Linear Gradients (linear-gradient())

A linear gradient is used to create a gradual transition between two or more colors along a straight line. It is set as the value for the background-image property.

1. Default Direction: to bottom

Red to Blue (Top to Bottom)

2. Direction Keywords

Green to Yellow (Left to Right)

3. Angles (45deg)

Black to Purple (45 degrees)

4. Color Stops (Smooth Transition)

By specifying a position (percentage or pixel) for a color, you control where the transition happens.

Light Blue (20%) to Green (80%)

5. Sharp Color Transitions / Border Effect (The Task)

Achieved by defining two colors with the exact same stop position, resulting in an immediate color change instead of a blend.

Multi-Colored Border Effect
