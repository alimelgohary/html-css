        opacity: 0.6;
        background-color: #6c757d; 
        pointer-events: none; /* 56-4: Disables all mouse/touch events */
    }

1. caret-color

Changes the color of the text input cursor (the blinking line) to match your design's color scheme.





2. pointer-events

Controls whether an element can be the target of mouse/touch events. Used to make an element completely non-interactive.

Clickable Link (Hover works)

When the link is disabled (grayed out), pointer-events: none; prevents the click event and the hover effect, making it completely non-interactive. This is useful for elements like buttons that should only be clicked once (as shown in the video's Hangman game example).
