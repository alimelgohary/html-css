Live Preview: Bouncing Dots

The staggered effect is achieved by applying different animation-delay values to each circle. By using alternate, the dots bounce smoothly between their start and end positions.

Key Concepts:

  • nth-child: Used to target specific circles within the container without adding extra classes to each.
  • Alternate Direction: This is crucial here; it makes the dot return to the bottom smoothly instead of jumping back instantly.
  • Relative Timing: Each delay (0.2s, 0.4s) creates the "wave" or "rhythm" effect.
