    /* Defining Animation with @keyframes */
    @keyframes change-color {
        from { background-color: red; }
        to { background-color: blue; }
    }

    /* Applying to Element */
    .box {
        animation-name: change-color;
        animation-duration: 3s;
    }
    @keyframes uneven-timing {
    0% {
        background-color: red;
    }
    80% {
        background-color: green; /* Spends 80% of the time transition to green */
    }
    100% {
        background-color: yellow; /* Spends only 20% moving to yellow */
    }
}

1. Simple Two-State Animation

Using from and to keywords to define a simple transition from a start state to an end state [00:01:29].

2. Multi-State Animation (Percentages)

Using percentages allows you to define multiple steps within the animation cycle [00:03:39]. Here, it cycles through Red, Blue, and Black.

Red → Blue → Black

3. Uneven Timing Distribution

You can control the speed of specific parts of the animation by adjusting percentage gaps [00:04:30]. This box stays Red/Green much longer than it stays Yellow.

Slow Transition

Key Concepts:

  • @keyframes: The rule used to define the animation sequence and its name [00:01:29].
  • animation-name: Links the element to the specific keyframe rule [00:02:10].
  • animation-duration: Defines how long one cycle of the animation takes to complete (e.g., 3s or 3000ms) [00:02:40].
