Block vs Inline

Block respects width/height/padding/margins, inline repsects right-left only

Block takes full width if no width specified

Block adds a line break, inline: no line break

Inline-Block allows elements side by side with (no line break) with respect to padding/margin

            div {
                background-color: gray;
                width: 200px;
                height: 100px;
                padding: 20px;
                margin: 10px;
            }
            .inline-span {
                background-color: greenyellow;
                padding: 20px;
                margin: 20px;
            }
            .inline-block-span {
                background-color: greenyellow;
                padding: 20px;
                margin: 20px;
                display: inline-block;
            }
Div => Block with width/height/padding/margin
Div => Block with width/height/padding/margin
Div => Block with width/height/padding/margin


Span => Inline Span => Inline Span => Inline
Hello div, this is broken because inline elements doesn't respect bottom/top padding/margin
