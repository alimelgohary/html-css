I am the Parent (Blue text, 24px, Solid Border)
I use inherit for my border [00:00:54].
I use color:initial, font-size: initial; (Back to black/default) [00:01:47].
I use unset (Inherits color, resets border) [00:04:31].
I use all: unset. I've lost all styles! [00:07:02]

Difference Summary:

  • inherit: "Give me whatever my parent has" [00:00:54].
  • initial: "Give me the value defined in the CSS specifications" (often ignores parent and browser defaults) [00:01:47].
  • unset: unset = initial or inherit. A smart hybrid. It prefers inheritance if the attribute is inheritable by default like "color" [00:04:31].
  • revert: The most "natural" reset. It goes back to how a clean browser would render the tag (like <h1> being bold again) [00:08:27].
