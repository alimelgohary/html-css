Selector Logic Summary:

  • Comma (,): OR logic. "Select A OR select B".
  • No Space (Chaining): AND logic on the *same* element. `div.active` means "a div AND it has class active". `.urgent.notify` means "has class urgent AND has class notify".
  • Space (Descendant): Ancestor/Descendant relationship. "Find B anywhere inside A".
  • Greater Than (>): Parent/Direct Child relationship. "Find B only if it is directly inside A".
