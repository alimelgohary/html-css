Key Takeaways:

  • :first-child is strict. The element must be the absolute first item in the parent [00:00:36].
  • :first-of-type is more flexible. It skips other tags to find the first instance of your specific tag [00:04:43].
  • :only-child requires zero siblings of any kind [00:08:16].
  • :only-of-type allows other siblings, as long as none of them are the same tag [00:09:05].
