Selector Logic Summary:

  • :not(): Think of this as the "Inverse" selector. It applies styles to everything that doesn't fit the rule inside [00:00:15].
  • :nth-child(n): Counts every child element regardless of type [00:01:45].
  • :nth-of-type(n): Filter by tag name first, then count the position [00:06:43].
  • -last-: Adding "last" to the name (e.g., :nth-last-child) flips the counting direction to start from the bottom [00:04:43].
