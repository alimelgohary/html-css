1. Content Injection and Stylistic Bars

The ::before element is used to create the absolute-positioned green bar on the left, and ::after is used to insert the text **(New)** at the end of the element.

User Information: Osama Elzero

2. Creating Shapes (Circle using border-radius)

The ::before element is given equal width and height, and then border-radius: 50%; is applied to turn it into a perfect circle.

This box has a circle icon generated entirely with CSS.

3. Creating Shapes (Arrow/Triangle using Borders)

By setting width and height to 0 and giving borders a color only on one side (with other borders transparent), a sharp triangular arrow shape can be generated.

This element has a green arrow pointing into it.
