1. Single Value (border-radius: 15px;)

A single value applies the same radius to all four corners (Top-Left, Top-Right, Bottom-Right, Bottom-Left).

15px Radius

2. Four Values (border-radius: TL TR BR BL;)

Values are set in a clockwise order, starting from the **Top-Left** corner: Top-Left, Top-Right, Bottom-Right, Bottom-Left.

5px 20px 40px 60px

3. Individual Corner Properties

For more control, you can use longhand properties like border-top-left-radius and border-bottom-right-radius.

Individual Radius

4. Creating a Perfect Circle or Oval

Applying border-radius: 50%; to a square element creates a perfect circle. Applying it to a rectangular element creates an oval (or ellipse).

Circle (50%)

5. Elliptical Corners (Horizontal / Vertical)

You can create elliptical or oval-like corners by separating two sets of values with a slash (/). The values before the slash define the horizontal radius, and the values after define the vertical radius.

Each corner can have 2 values border-top-left-radius: 50px 20px;

50px / 10px
top right: 50px 20px
