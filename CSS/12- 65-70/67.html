1. Basic Translation using `translate(x, y)`

The translate(x, y) function moves an element from its original position. The first value (50px) is the shift along the X-axis (horizontal), and the second value (20px) is the shift along the Y-axis (vertical). Positive values move right and down.

2. Horizontal Movement using `translateX(x)`

To move an element only horizontally, use translateX(x). This is a shorthand for translate(x, 0) and makes the intention clearer.

Hover to TranslateX [00:59]

3. Vertical Movement using `translateY(y)` (Percentage)

The translateY(y) function moves an element only vertically. When using a percentage, it is relative to the element's *own* height. -50% moves the element up by half of its own height.

Hover to TranslateY (-50%) [01:05]

4. Centering with `position` and `translate()`

This is a powerful technique for perfect centering. By setting top: 50%; and left: 50%;, the element's top-left corner is positioned at the container's center. Then, transform: translate(-50%, -50%); shifts the element back by exactly half of its own width and height, positioning its true center at the container's center. This is superior to using margin offsets because it is size-independent.
