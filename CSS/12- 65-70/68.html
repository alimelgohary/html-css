1. Skewing Along the X-Axis (`skewX()`)

The skewX() function tilts an element horizontally by a specified angle. Positive values tilt the top of the element to the right, and negative values tilt it to the left. The example uses 20deg to slant the element.

Hover to SkewX (20deg) [00:35]

2. Skewing Along the Y-Axis (`skewY()`)

The skewY() function tilts an element vertically. Positive values tilt the right side of the element downwards, and negative values tilt it upwards. The example uses 15deg for vertical slanting.

Hover to SkewY (15deg) [00:45]

3. Combined Skewing (`skew(x, y)`)

The shorthand skew() function accepts two angle values: the first for the X-axis and the second for the Y-axis. This allows for a combined, compound slanting effect, like skew(10deg, -5deg).
