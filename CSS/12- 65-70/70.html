1. Default Origin (`center center`)

By default, transformations like rotation occur around the element's exact center (50% 50%). When you rotate, the box spins around its middle point.

Rotate (Center) [04:07]

2. Top-Left Origin (`top left`)

Setting the origin to top left (or 0% 0%) moves the pivot point to the upper-left corner. The box will swing around this fixed corner during rotation.

Rotate (Top-Left) [08:07]

3. Bottom-Right Origin (`bottom right`)

Setting the origin to bottom right (or 100% 100%) moves the pivot point to the lower-right corner. All transformations now use this corner as the stationary anchor.

Rotate (Bottom-Right) [06:03]

4. Custom Origin using Percentages

You are not restricted to keywords. You can specify exact coordinates using percentages or length units. 20% 80% places the origin point near the bottom-left, but offset slightly into the box's area.
