1. Uniform Scaling using `scale(N)`

The scale() function scales an element uniformly across both the X and Y axes by a single factor (N). A value of 2 doubles the size of the element horizontally and vertically upon hover.

2. Horizontal Scaling with `scaleX(N)`

The scaleX() function scales the element only along the horizontal (X) axis. Using scaleX(2) doubles the element's width while keeping its height the same.

3. Vertical Scaling with `scaleY(N)`

The scaleY() function scales the element only along the vertical (Y) axis. Using scaleY(2) doubles the element's height while keeping its width the same.

4. Non-Uniform Scaling using `scale(Nx, Ny)`

You can specify different scaling factors for the X and Y axes by passing two arguments to the scale() function. This example doubles the width (2) and halves the height (0.5).

5. Flipping the Element using Negative Scale

Using a negative value for the scaling factor, such as scale(-1), flips the element along the specified axis (or both axes if a single value is provided), reversing its direction.
