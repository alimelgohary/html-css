1. Using `matrix()` for Uniform Scaling (ScaleX, ScaleY)

The `matrix(a, b, c, d, e, f)` function represents a 2D transformation. The values `a` and `d` control scaling. Setting them both to `2` results in a uniform double-size scaling effect, identical to `scale(2)`.

2. Using `matrix()` for Translation (TranslateX, TranslateY)

The last two values, `e` and `f`, control translation (movement). Setting `e` and `f` to `30` moves the element 30 units right and 30 units down, identical to `translate(30px, 30px)`. The first four values are set to the default identity matrix (1, 0, 0, 1) to prevent scaling or skewing.

4. Combined Matrix Example

This final example uses `matrix(1.2, 0, 0.3, 1, 20, 10)` to apply a 1.2x scale on X (`a=1.2`), a moderate skew on X (`c=0.3`), and a translation of (20, 10) on the last two values. This highlights how `matrix()` can perform multiple transforms with a single property declaration.
