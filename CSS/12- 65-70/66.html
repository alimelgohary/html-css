        /* Combination of multiple transforms */
        transform: rotate(5deg) scale(1.2);
        opacity: 0.7;
    }

1. Basic 2D Rotation (`rotate(Ndeg)`)

The rotate() function turns the element clockwise around its center point by the specified degree value. This example rotates the box by 45 degrees when hovered.

Hover to Rotate (45deg)

2. Counter-Clockwise Rotation (Negative Value)

A negative degree value is used to rotate the element in the counter-clockwise direction. This demonstrates a -30 degree rotation.

Hover to Rotate (-30deg)

3. Rotation Around the X-Axis (`rotateX()`)

The rotateX() function rotates the element around the horizontal X-axis, making it appear to flip or spin into/out of the screen. A 180-degree rotation flips it completely.

Hover to RotateX (180deg)

4. Rotation Around the Y-Axis (`rotateY()`)

The rotateY() function rotates the element around the vertical Y-axis, making it appear to turn like a page. A 90-degree rotation makes the element disappear edge-on.

Hover to RotateY (90deg)

5. Combining Transforms

Multiple transform functions can be applied to a single element. This box is rotated 15 degrees and simultaneously scaled up by 1.2, showing how transformations can be chained.

Hover to Combine (Rotate & Scale)
