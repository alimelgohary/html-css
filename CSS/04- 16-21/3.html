Text align: aligns text regardless of its direction RTL or LTR

            
            .one {
                text-align: right;
            }
            .two {
                text-align: left;
            }
            .three {
                text-align: center;
            }
            .four {
                direction: rtl;
                text-align: center;
            }
            .five img {
                vertical-align: top;
            }
