overflow: scroll: make a scroll (even if no overflow)

overflow: auto: scroll but better (Scroll is visible only if the content overflows)

overflow: hidden

You can control overflow-x, y separately (Can be useful in responsive design)

overflow-x: hidden, overflow-y: hidden

            .scroll {
                overflow: scroll;
            }
            .auto {
                overflow: auto;
            }
            .hidden {
                overflow: hidden;
            }
            .hiddenx {
                overflow-x: hidden;
            }
            .hiddeny {
                overflow-y: hidden;
            }
