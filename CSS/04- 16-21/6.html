Demonstrating how long text is truncated using text-overflow: ellipsis; The full text becomes visible on hover, and the title attribute shows the text on mouse hover. 

    
      .box {
          width: 250px;
          padding: 10px;
          border: 1px solid #ccc;
          white-space: nowrap;
          overflow: hidden;
          text-overflow: ellipsis;
      }

      /* On hover, show the overflowed text */
      .box:hover {
          overflow: visible;
          background-color: #f7f7f7;
      }
This is a very long line of text that normally gets truncated with an ellipsis, but becomes fully visible on hover.
