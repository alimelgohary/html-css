    /* Container has align-items: stretch; */
    .flex-container {
        height: 150px; 
        align-items: stretch;
    }

    /* 1. Default/Auto */
    .self-auto {
        align-self: auto; /* Inherits stretch (must not have explicit height) */
    }

    /* 2. Item at the Top */
    .self-start {
        align-self: flex-start;
    }

    /* 3. Item at the Bottom */
    .self-end {
        align-self: flex-end;
    }

    /* 4. Item Centered */
    .self-center {
        align-self: center;
    }

    /* 5. Item Baseline */
    .self-baseline {
        align-self: baseline;
    }

1. Flex Item: align-self

Overrides the container's align-items property for a single flex item. The container's default is stretch.

2. Recommended Flexbox Practice

The video encourages the use of online resources to master Flexbox. Suggested resources include:

  • Flexbox Froggy: A simple game to learn container properties (flexboxfroggy.com).
  • Flexbox Zombies: A narrative-driven, deeper dive into all Flexbox properties (mastery.games/flexboxzombies).
