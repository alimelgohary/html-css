    /* All examples use 'display: flex' */

    /* 1. Default Behavior */
    .ex-start {
        justify-content: flex-start;
    }

    /* 2. Items at the End */
    .ex-end {
        justify-content: flex-end;
    }

    /* 3. Items Centered */
    .ex-center {
        justify-content: center;
    }

    /* 4. Space Between Items */
    .ex-between {
        justify-content: space-between; 
    }

    /* 5. Space Around Items */
    .ex-around {
        justify-content: space-around;
    }

    /* 6. Space Evenly */
    .ex-evenly {
        justify-content: space-evenly;
    }

1. Justify Content: flex-start

Packs flex items toward the start of the main axis. This is the default behavior.

Items are evenly distributed in the line. The first item is on the start line, and the last item is on the end line.

5. Justify Content: space-around

Items are evenly distributed with equal space around them. Note that the space at the ends is half the space between the items.

6. Justify Content: space-evenly

Items are distributed such that the spacing between any two adjacent items, and the space before the first and after the last, is exactly the same.
