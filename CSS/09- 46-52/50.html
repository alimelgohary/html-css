    /* 1. Flex Grow */
    .ex-grow .flex-item {
        flex-grow: 1; 
    }
    .ex-grow .grow-3 {
        flex-grow: 3; /* Grows 3x more than others */
    }

    /* 2. Flex Shrink */
    .ex-shrink .flex-item {
        width: 150px; /* Base size is too large for container */
        flex-shrink: 1; 
    }
    .ex-shrink .no-shrink {
        flex-shrink: 0; /* Prevents item from shrinking */
    }

    /* 3. Order */
    .ex-order .flex-item {
        order: 0; /* Default */
    }
    .ex-order .order-1 {
        order: 5; /* Moves to the end */
    }
    .ex-order .order-2 {
        order: -1; /* Moves to the beginning */
    }
    .ex-order .order-3 {
        order: 2; 
    }

1. Flex Item: flex-grow

Controls how much an item should grow to fill the remaining space. Default is 0 (no growth). All items have a base width of 80px and flex-grow: 1, except for Box 3 which has flex-grow: 3.

Box 1 (Grow: 1)
Box 2 (Grow: 1)
Box 3 (Grow: 3)
Box 4 (Grow: 1)
Box 5 (Grow: 1)

2. Flex Item: flex-shrink

Controls how much an item should shrink when the container is smaller than the sum of item widths. Default is 1 (shrinks equally). All items have a base width of 150px, but Box 3 is set to flex-shrink: 0.

Box 1 (Shrink: 1)
Box 2 (Shrink: 1)
Box 3 (Shrink: 0)
Box 4 (Shrink: 1)

3. Flex Item: order

Sets the sequential order of the flex item within the container. Items with a lower number appear first. Default order is 0. HTML order: 1, 2, 3, 4, 5.

Box 1 (Order: 0)
Box 2 (Order: -1)
Box 3 (Order: 2)
Box 4 (Order: 5)
Box 5 (Order: 0)
