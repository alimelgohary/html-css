    /* 1. Flex Basis */
    .ex-basis .item-51-1 {
        flex-basis: 100px; 
        flex-grow: 1; /* Grows from 100px */
    }
    .ex-basis .item-51-2 {
        flex-basis: 40%; /* Takes 40% of container space */
        flex-grow: 0; 
    }
    .ex-basis .item-51-3 {
        flex-basis: auto; /* Uses explicit 'width' (60px) */
        width: 60px; 
        flex-grow: 1;
    }

    /* 2. Flex Shorthand (grow shrink basis) */
    .ex-shorthand .item-51-4 {
        flex: 1; /* Expands to 1 1 0% */
    }
    .ex-shorthand .item-51-5 {
        flex: 2 1 auto; /* 2x growth, 1x shrink, size based on 80px width */
        width: 80px;
    }
    .ex-shorthand .item-51-6 {
        flex: none; /* Expands to 0 0 auto - fixed size */
        width: 150px;
    }

    /* 3. Inline-Flex */
    .ex-inline-flex {
        display: inline-flex; /* Container acts inline, but contents are flex */
    }

1. Flex Item: flex-basis

Sets the initial main size of the flex item before any `grow` or `shrink` calculation.

51-1 (Basis: 100px, Grow: 1)
51-2 (Basis: 40%, Grow: 0)
51-3 (Basis: auto, Width: 60px, Grow: 1)

2. Flex Item: flex Shorthand

Shorthand for flex-grow, flex-shrink, and flex-basis.

51-4 (flex: 1)
51-5 (flex: 2 1 auto)
51-6 (flex: none)

3. Display: inline-flex

Makes the container behave like an **inline** element with respect to surrounding content (it only takes the width of its content), but its direct children are still laid out using the flex model.
