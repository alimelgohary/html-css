        flex-flow: row-reverse wrap;
    }

        width: 30%; 
    }

1. Basic Flex Container (display: flex)

The display: flex property turns the container into a flex context. By default, items align horizontally (row) and attempt to fit on a single line (nowrap).

3. Wrap (flex-wrap: wrap)

When set to wrap, flex items that exceed the container's space will wrap to the next line instead of shrinking to fit on one line.
