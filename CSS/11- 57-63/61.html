1. Defining Column Span with Explicit Start and End Lines

This method explicitly defines the starting and ending grid lines for an item. The video shows that setting grid-column-start: 1; and grid-column-end: 5; makes the item occupy 4 columns, as the end line is exclusive in this context.

2. Shorthand for Start Line and Column Span (`/ span N`)

The shorthand property grid-column is used here to define both the starting line and the number of columns the item should span (span 4). Setting grid-column: 2 / span 4; will start the item at line 2 and make it take up the next four column tracks.

3. Spanning Multiple Rows with Shorthand (`grid-row`)

Similar to columns, the grid-row shorthand can be used to control vertical placement. Setting grid-row: 1 / span 2; makes the item start at the first row line and expand downward to cover two full rows, demonstrating the power of Grid for complex layouts.
