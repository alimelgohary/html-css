2. Defining Grid Structure and Assigning Elements with `grid-area`

The primary method for advanced layouts is to first define named areas on the parent container using grid-template-areas. Here, we define a layout with a split header (logo/links) and a full-width content section below. The child items are then assigned to these named areas using the grid-area property on the child elements.
