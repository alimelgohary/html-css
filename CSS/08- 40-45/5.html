1. Vertical Collapse: Largest Margin Wins

The gap between the boxes is **25px**, not 35px (10px + 25px). The margins collapse, and the largest value wins.

Box A (margin-bottom: 10px)
Box B (margin-top: 25px)

2. Preventing Collapse (Parent and First Child)

If the parent element has a border, padding, or content separating the parent's top edge from the child's top margin, the collapse is prevented. Here, the border prevents the child's margin-top from moving outside the parent.

Box C (margin-top: 20px)
