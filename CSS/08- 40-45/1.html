1. Basic Box Shadow Syntax

The core syntax for box-shadow is: [H-offset] [V-offset] [Blur-radius] [Color].

Example: box-shadow: 10px 10px 5px #888888;

Basic Shadow

2. Including Spread Radius

The optional **Spread Radius** is placed before the color: [H] [V] [Blur] [Spread] [Color]. A positive spread value expands the shadow, while a negative value shrinks it.

Example: box-shadow: 5px 5px 10px 3px rgba(0, 0, 0, 0.5);

Shadow with Spread

3. Creating an Inner Shadow (Inset)

The optional **inset** keyword makes the shadow an internal one, giving the appearance of a recessed or pressed element. It should be placed at the very beginning of the declaration.

Example: box-shadow: inset 0 0 10px red;

Inset Shadow

4. Applying Multiple Shadows

You can apply multiple shadows to a single element by separating each full shadow declaration with a **comma** (,). The first shadow listed is applied on top.

Multiple Shadows
