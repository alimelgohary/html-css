    :root {
        --main-color: #007bff;
        --main-padding: 10px;
    }
    
    .global-color {
        background-color: var(--main-color, red); 
        border-color: var(--main-color);
    }
    
    .global-padding {
        padding: calc(var(--main-padding) + 15px);
        background-color: #28a745;
    }
    
    .local-scope {
        --scope-color: #ffc107;
        background-color: var(--scope-color);
        color: black;
    }

1. Global Variable Usage

Variables are defined in the :root selector and accessed anywhere using var(--variable-name).

Background: var(--main-color)

2. Variables in CSS Calculations

You can use the calc() function to perform mathematical operations using your variables, allowing for dynamic sizing.

Padding: calc(var(--main-padding) + 15px)

3. Local Variable Scope

A variable defined on an element (like .local-scope) is only available to that element and its descendants, overriding any global variable with the same name.

Background: var(--scope-color)
