1. Basic Transition (Implicit 'All')

Setting only transition-duration allows all changeable properties (width and background-color) to animate smoothly over 2 seconds.

Hover Me

2. Transitioning Specific Properties (transition-property)

For better performance, you should specify exactly which property to animate. Here, only the width animates, while the background color changes instantly.

Hover Me

3. Adding a Delay (transition-delay)

The transition waits for 1 second (transition-delay: 1s;) before starting the 2-second animation.

Hover (Wait 1s)

4. Multiple Properties with Different Durations

You can comma-separate properties to give them distinct durations. The width takes 2s, while the height takes only 1s.

Hover Me

5. The Shorthand Property

The transition shorthand follows the order: property | duration | timing-function | delay.
